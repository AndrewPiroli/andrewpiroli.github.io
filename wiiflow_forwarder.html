Installing/Autobooting Wiiflow via Priiloader

WiiFlow is weird, it does not like to be directly launched by Priiloader.
So you need to use a "forwarder dol" to load, which will then load the real thing.
You can find the forwarder dol here
Rename it to 'boot.dol', create a new folder in your 'apps' folder alongside your other homebrew, name it 'wiiflow_forwarder' and place the file you downloaded above in there.
Now when you enter Priiloader, go to Load/Install File and select 'wiiflow_forwarder', it will now be installed and available to be used in launching from Priiloader or autobooting.
The forwarder tries to load the path sd:/apps/wiiflow/boot.dol, if that's not there, it will try to load the same path from USB, the folder name MUST exactly match.
