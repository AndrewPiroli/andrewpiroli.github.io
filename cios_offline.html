Install cIOS on a Wii with no internet connection

The d2x installer needs to connect to Nintendo's update servers to get certain files in order to function. If your Wii is not connected, you need to download the files using a PC.

Method #1 - Automatic Download Script

Contributed by MethodOrMadness

Download the zip containing the script here

  1. Extract the files
  2. Open the folder
  3. Run "cios_offline.bat" by double clicking
  4. Follow the on-screen instructions
  5. When finished, you will see a folder "cios_base" with 3 WAD files.
 

Method #2 - Manual Download

  1. Download and open NUS Downloader
  2. Click "Database", hover over "IOS"
  3. Go to "IOS57" then click "v5918"
  4. At the bottom of the program ensure "Pack WAD" is checked, and "Patch IOS" is unchecked.
  5. Click "Start NUS Download"
  6. Repeat the previous steps for IOS56 v5661 and IOS38 v4123
  7. The WADs are saved in the "titles" folder in the same location NUS Downloader is saved, you may have to dig through mutliple subfolders to find all the WADs

Once you have the WADs, place them in the root of your SD card or USB storage device, whichever you use to hold homebrew. d2x installer will see them and use them automatically.

This WAD selection matches up with wii.guide's cIOS installation at the time of writing, if the recommended guide changes or you are deviating from the guide for some reason, d2x installer will tell you the name of the WAD it wants, which includes the base IOS and revision. Simply follow method #2 above with the base and revision needed.
